# Development howto

## Build

```
$ npm install # install dependencies
$ grunt firefox
$ grunt chrome
```

## Test

```
$ npm test
```

## Release a beta version

```
$ npm version preminor
$ grunt firefox
$ grunt chrome
```

### Release second beta version

```
$ npm version prerelease
$ grunt firefox
$ grunt chrome
```
